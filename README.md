# 🌤️ Бот погоды
## Описание проекта
Телеграм-бот, который показывает актуальную погоду в вашем местоположении. При отправке геолокации - бот вернет температуру, описание погоды, влажность. 

Добавлена кнопка "О проекте", чтобы узнать о чем он, и кто его создал

## Информация об использованных технологиях
- **Python 3.x**
- Библиотека **pyTelegramBotAPI (telebot)** — для взаимодействия с Telegram
- Библиотека **requests** — для запросов к OpenWeatherMap API
- **OpenWeatherMap API** — источник данных о погоде

## Инструкция к запуску
1. Установите зависимости
```
pip install pyTelegramBotAPI requests 
```
2. Установите свои ключи
```
- TOKEN = '...' → на ваш токен от @BotFather
- API_KEY = '...' → на ваш ключ от OpenWeatherMap
```
3. Запустите бота
```
python main.py
```
4. Откройте Телеграм
Откройте своего бота и пропишите
```
/start
```
И отправьте свое местоположение через кнопку "Получить погоду"
